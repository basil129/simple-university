# How to use git and github

change files->
1. after changing files you have to commit the changes to local repository
2. to commit changes from vs code id use the left icon ctl+shift+g
3. for each commit you need message
4.write the message in the text box and click commit. message should be meaningful
5.after commiting the changes are stored in local repository not git hub
6.to apply changes to github repository[upstream] you have to push the changes to github by clicking the push icon
